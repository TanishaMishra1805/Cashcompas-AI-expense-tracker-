 CashCompas 
1. CashCompas is an AI‑enabled personal finance tracker designed to simplify expense management and smart investment planning.  
2. It helps users track spending, set savings goals, and get personalized recommendations for better financial decisions.

 Features
1. Expense Tracking : Daily, category‑wise and recurring expense analysis  
2. AI‑Driven Savings Suggestions : Personalized recommendations to reduce overspending  
3. Budget & Goal Management : Set savings goals and monitor progress visually  
4. Investment Guidance : Risk‑aware insights for smarter financial decisions  
5. Interactive Dashboards : Charts, heatmaps and reports powered by Pandas & Matplotlib  
6. Cloud Deployment : Dockerized backend with AWS integration for scalability  

 Tech Stack
1. Backend: Python, Flask, SQLite, Docker
2. Frontend: HTML, CSS
3. Data Analysis: Pandas, Matplotlib
4. Cloud & Deployment: AWS (planned deployment with CI/CD pipeline)

 Installation

1. Clone the repository :
```bash
   git clone https://github.com/TanishaMishra1805/CashCompas.git
   cd CashCompas
```

2. Install dependencies :
```bash
   pip install flask pandas matplotlib
```
